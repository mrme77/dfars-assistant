# Workflow: Answer DFARS Question

1. Classify the user question:
   - exact clause or section lookup
   - applicability question
   - definition question
   - comparison question
   - general explanation
2. Retrieve candidate sections using exact identifiers, keyword search, and semantic search.
3. Re-rank candidates against the user question.
4. Build the answer context:
   - assistant rules
   - citation policy
   - user question
   - retrieved section metadata
   - original source text for top sections
5. Answer with citations and applicability limits.
