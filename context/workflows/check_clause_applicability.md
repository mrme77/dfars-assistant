# Workflow: Check Clause Applicability

1. Identify the clause or topic.
2. Retrieve prescription sections and the clause text.
3. Check whether the user provided:
   - acquisition type
   - contract type
   - commercial product/service status
   - dollar threshold if relevant
   - covered information or supply-chain facts if relevant
4. If facts are missing, ask for them.
5. Provide a conditional answer with cited DFARS sections.
